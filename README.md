Playwright Automation Test – Shopping Cart
Automated testing of the shopping cart functionality in a web application using Playwright.

Project Overview
This project focuses on ensuring the reliability and correctness of shopping cart interactions. Automated tests cover core features such as:
Adding and removing products from the cart
Verifying correct display of cart elements
Testing promotional offers and discounts (e.g., buy 1 get 3)
Validating cart behavior under various user interactions
The goal is to maintain high-quality user experience and ensure all cart functionalities work as expected.

Getting Started
Clone the repository: git clone https://github.com/VladyslavKhmara11/playwright-automation-test.git
Install dependencies: 'npm i' and 'npx playwright install'
Run tests: npx playwright test
